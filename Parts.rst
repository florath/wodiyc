=======================

direction (parallel to the Z-axis):

   AAAAAAAA
   AA
   AA

   AA

(0,0) is on the left bottom.


SupportBearing
==============
Use 40mm x 720mm wood. Use 8mm tool for cleaning surface.
Use 3.125mm tool for holes.
Place wood parallel to Y-axis.
